## מבנה האתר

- `index.html` — מבנה דף הבית והטבלאות
- `assets/css/style.css` — עיצוב בלבד
- `assets/js/app.js` — טעינה, חיפוש וסידור טבלאות
- `data/trainings.json` — נתוני השתלמויות
- `data/meetings.json` — נתוני מפגשים
- `data/search-index.json` — אינדקס חיפוש
- `data/missing-data.json` — נתונים חסרים, כרגע ריק
