# כללי מערכת hishtalmut

## כלל עליון
דף הבית מציג רק מידע לפי הטבלאות שנמסרו על ידי יניב.

## מבנה חובה
- HTML ב-index.html
- CSS ב-assets/css/style.css
- JavaScript ב-assets/js/app.js
- נתונים ב-data/*.json

## אסור להציג
- מידע דמו
- טקסט שיווקי
- כפתורי פעולה שאינם מידע מהטבלאות
- קבצים אם לא נמסרו קבצים
- שמות מרצים אם לא אומתו בוודאות
- אופן למידה אם לא אומת בוודאות
- מספר השתלמות מומצא

## מותר להציג
- מספר השתלמות אם מופיע
- אם מספר ההשתלמות לא מופיע: חסר
- תאריך
- יום
- שעות
- שעות אקדמיות
- מספר מפגש
- נושא
- סיכומי ספירה וחיבור שנובעים ישירות מהטבלאות
